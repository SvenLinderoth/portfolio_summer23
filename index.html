Trip-it

In this project we were three people cooperating by using the schools API. This API stored information that we had to use for our application.

We had an assigned focused group and task, and were somewhat limited in what to do with the API. The focus of this project was to gain a greater knowledge in how you can work using application protocol interfaces.

This project overall helped me understand how an API can be used and why it could be beneficial during development. Later on in my second year of University, (this project is from my first year) , i then created my own API in PHP that dealt with connection to a database, and this certainly helped me understand how the data is retrieved and used.

Main focus: Group-Project, API
